Comment ajouter votre photo au CV - Méthode Simple

Nous avons créé une solution temporaire pour afficher vos initiales sur votre CV. Voici comment ajouter votre vraie photo si vous le souhaitez.

Étape 1: Modifiez directement le code HTML

Ouvrez le fichier index.html dans un éditeur de texte (comme Notepad ou Visual Studio Code)

Étape 2: Trouvez la section de l'image

Cherchez cette partie du code (vers la ligne 30):

<div class="profile-image">
    <img src="https://ui-avatars.com/api/?name=Amal+Chalhi&background=e6f3ff&color=3498db&size=140&bold=true&font-size=0.4" alt="Amal Chalhi">
</div>

Étape 3: Remplacez par votre photo

Si vous avez accès à un service d'hébergement d'images (comme Google Drive, Dropbox ou Github), téléchargez votre photo et obtenez un lien direct vers l'image.

Ensuite, remplacez le code par:

<div class="profile-image">
    <img src="VOTRE_LIEN_DIRECT_VERS_L_IMAGE" alt="Amal Chalhi">
</div>

Remplacez VOTRE_LIEN_DIRECT_VERS_L_IMAGE par l'URL de votre image.

Note: Si vous n'avez pas de lien d'image en ligne, vous pouvez:

  • Utiliser un service comme ImgBB pour télécharger gratuitement votre photo
  • Ou garder la version actuelle avec vos initiales qui est également professionnelle
