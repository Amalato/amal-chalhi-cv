Comment ajouter votre photo au CV

Note importante: Votre photo doit être recadrée pour ne montrer que votre tête et vos épaules pour un résultat professionnel.

Étapes à suivre :

  1. Recadrez votre photo - Utilisez un logiciel comme Paint, Photoshop ou un outil en ligne pour recadrer votre photo afin qu'elle montre uniquement votre tête et vos épaules.
  2. Redimensionnez la photo - Idéalement, votre photo devrait avoir une taille carrée (par exemple 400x400 pixels).
  3. Enregistrez la photo - Enregistrez-la sous le nom "profile.jpg" dans le dossier suivant:
    C:\Users\hp\cv-project\
  4. Actualisez votre CV - Ouvrez votre CV dans le navigateur et actualisez la page pour voir la photo.

Le CSS a été configuré pour recadrer automatiquement la photo afin de se concentrer sur votre visage et vos épaules. Si le cadrage ne vous convient pas, vous pouvez modifier la valeur object-position dans le fichier CSS.
